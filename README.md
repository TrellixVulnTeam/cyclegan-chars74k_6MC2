# cyclegan-chars74k
Chars74Kデータセットを使用したCycleGANによる手書き文字からコンピュータ文字へのマッピングのテストです。

## 実行方法
`python train.py`  
はじめて実行した時はデータセットをダウンロードするため、しばらく時間がかかります。
